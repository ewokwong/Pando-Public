Simple MVP of Pando using:
- React for Frontend
- Node.js for Backend
- Express for Server
- MongoDB for DB
- JWT Tokens for authentication

Setup
- Pando
-   Run npm install

- frontend-next
-   npm install
-   npm run build

- backend
-   npm install
-   npm run build

To Start
- backend
-   brew services restart mongodb-community
-   npm start

-   frontend
-     npm run dev
